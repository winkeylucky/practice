### js自带方法
Number对象中自带进制转换的方法```toString(n)```, 可以把 10进制 转换为指定进制
```js
29.toString(2);  //报错!
29..toString(2);  // "11101" 这还差不多
(29).toString(2);  // "11101" 这样更好

(29).toString(16); // "1d"
```

把指定进制转换为10进制哩?
```js
parseInt('1111', 10); // 1111
parseInt('1111', 2); // 15
parseInt('1111', 16); // 4369
```
